== Description ==

Automatically show a mobile popout for navigation on mobile devices.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the ‘mp-menu’ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the colours and pop-out location not he “Appearance” > “Customize” screen in WordPress under “MP Menu”.
